## Overview

Crimson is a modern, sleek color theme designed for optimal readability and visual comfort. It features a deep obsidian background (#0e0c0d) with subtle ruby undertones for the dark theme and clean whites for the light theme, paired with vibrant crimson accents and carefully selected complementary colors following color theory principles.

The theme comes in two variants:
- **Crimson Dark** â€” Off-black backgrounds with vibrant, readable colors
- **Crimson Light** â€” Clean whites with rich, contrasting colors

## Color Theory & Design Principles

### 1. **Deep Obsidian with Ruby Warmth**
The dark theme uses `#0e0c0d` for panels and `#16131a` for the editorâ€”deep, rich blacks with subtle ruby undertones. This creates a luxurious, modern feel while reducing eye strain. The distinct panel and editor backgrounds provide clear visual separation, making navigation intuitive.

### 2. **Vibrant Yet Readable**
Colors are carefully selected to be vibrant and eye-catching while maintaining excellent readability. Each color has sufficient contrast against both light and dark backgrounds.

### 3. **Complementary Color Scheme**
The theme follows color theory principles with a primary red (crimson) complemented by:
- **Green** (complementary) for strings and success states
- **Blue** for functions and info states
- **Yellow/Gold** for types and warnings
- **Cyan** for enums and special elements
- **Violet/Purple** for preprocessor directives

### 4. **Semantic Consistency**
Colors maintain consistent meaning throughout the theme:
- Red shades: Keywords, errors, deletions, tags
- Green shades: Strings, success, additions
- Blue shades: Functions, methods, information
- Yellow/Gold: Types, constants, warnings
- Cyan: Enums, labels, escapes, renamed items

### 5. **Hierarchical Clarity**
Multiple shades create clear visual hierarchy:
- Background layers progress from darkest to lightest
- Text colors indicate importance (primary, muted, subtle, disabled)
- Syntax colors distinguish between different code elements

### 6. **Accessibility First**
All color combinations meet or exceed WCAG contrast guidelines for readability. Text remains legible in all contexts.

### 7. **Modern Aesthetics**
Clean, neutral backgrounds let the vibrant syntax colors shine without overwhelming the interface. The theme feels contemporary and professional.

## Installation

1. Place `crimson.json` in your Zed themes directory:
   - **macOS/Linux**: `~/.config/zed/themes/Crimson/`
   - **Windows**: `%APPDATA%\Zed\themes\Crimson\`

2. Open Zed's Command Palette (`Cmd/Ctrl + Shift + P`)

3. Search for "Select Theme"

4. Choose **Crimson Dark** or **Crimson Light**
